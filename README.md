# W2NVShellMessage
A quote from [Welcome to Night Vale](https://www.welcometonightvale.com/) to show when bash is launched.

## Installation
```bash
$ pip install w2nvshellmessage
```

## Run tests
```bash
$ pytest
```

## Initialization 
```bash
$ w2nv-init
```

## Usage
```bash
$ w2nv
```

## .bashrc
```bash
$ echo 'w2nv' >> ~/.bashrc
```
